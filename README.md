# SASE Decision Studio

Aplicacion estatica para valorar visualmente proveedores SASE/SSE:

- Zscaler
- Netskope
- Palo Alto Networks
- Fortinet
- Cisco

Incluye scoring ponderado, casos de uso imprescindibles, matriz de riesgo, ENS, cifrado, implantacion/provision, posicion on-premise/soberana y casos de exito publicos.

## Publicacion

Esta app esta preparada para GitHub Pages desde la rama `main`, carpeta raiz.
